# 🎓 Espaço Maais — Site Institucional

Site institucional do **Espaço Maais — Espaço Pedagógico**, empresa de reforço escolar e alfabetização com mais de 9 anos de experiência.

🌐 **Site publicado:** [espacomaais.netlify.app](https://espacomaais.netlify.app)

---

## 📋 Sobre o projeto

O Espaço Maais oferece:
- Alfabetização para crianças a partir de 4 anos
- Reforço escolar do Fundamental I ao Ensino Médio
- Alfabetização de Jovens e Adultos (EJA)
- Informática (Word, Excel, PowerPoint, Canva)
- Acompanhamento pedagógico personalizado

Este site foi desenvolvido como projeto acadêmico e entregue como solução real para o cliente.

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização, responsividade e animações |
| JavaScript (vanilla) | Menu mobile, carrossel de fotos |
| Google Fonts | Tipografia (Baloo 2 e Nunito) |
| Base64 | Imagens incorporadas diretamente no HTML |

---

## 📁 Estrutura do projeto

```
ajuda-maais/
└── index.html       # Arquivo principal (HTML + CSS + JS + imagens em Base64)
```

O projeto é um **único arquivo HTML autocontido** — sem dependências externas de arquivos locais, facilitando a hospedagem em qualquer plataforma estática.

---

## 🗂️ Seções do site

- **Hero** — Apresentação, foto da professora e estatísticas (9+ anos, 500+ alunos, 300+ avaliações)
- **O que ensinamos** — Cards com os serviços educacionais
- **Nossos Serviços** — Detalhamento dos serviços
- **Nossa Estrutura** — Fotos do espaço físico
- **Nossos Momentos** — Galeria de fotos em carrossel (20 fotos)
- **Depoimentos** — Avaliações de clientes
- **Localização** — Endereço e mapa
- **Contato** — WhatsApp e canais de atendimento

---

## 🚀 Como rodar localmente

Não precisa de servidor ou instalação. Basta:

1. Baixar o arquivo `index.html`
2. Abrir no navegador (Chrome, Firefox, Edge etc.)

---

## ☁️ Como publicar (deploy)

O site está hospedado no **Netlify** (hospedagem gratuita para sites estáticos).

Para publicar uma nova versão:

1. Acesse [app.netlify.com](https://app.netlify.com)
2. Entre no projeto `espacomaais`
3. Vá em **Implantações**
4. Arraste o arquivo `index.html` para a área de upload
5. Aguarde o deploy automático
